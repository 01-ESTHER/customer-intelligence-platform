# Customer Intelligence Platform

## Project Overview

This project is an end-to-end Customer Intelligence Platform built using Python, SQL, PostgreSQL, and Machine Learning.

The objective is to transform raw customer data into actionable business insights through data engineering, exploratory data analysis, feature engineering, predictive modelling, and interactive dashboards.

The project follows industry-standard data science and software engineering practices, including version control, modular code structure, and reproducible workflows.
